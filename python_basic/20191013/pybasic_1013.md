4. set, tuple 은 언제쓰지? 개념이 헷갈리고, 자주 안써봤는데 괜찮을까?
- tuple은 값을 바꿀 수 없고 list는 값을 바꿀 수 있다.
- 함수에서 값 여러개를 한 번에 return 하는 경우 tuple로 만들어서 return 해준다.
- set은 [] 형태가 아닌 {} 형태로, indexing이 불가능하다.
```python
a = {1, 2, 3}
a[0] # error
```
- 집합의 형태로서 합집합, 차집합 등의 연산을 수행할 때 사용할 수 있다.
```python
a = {1, 2, 3}
b = {2, 3, 4}

a|b # {1, 2, 3, 4}
a-b # {1}
b-1 # {4}
```
